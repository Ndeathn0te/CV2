# Armando Antonio González Tapiador - Curriculum Vitae

Este es un proyecto de Curriculum Vitae (CV) interactivo, moderno y profesional, construido con **React 19**, **Vite** y **Tailwind CSS 4**. Está diseñado para ser altamente responsivo, optimizado para la impresión en PDF y fácil de desplegar en plataformas como Cloudflare Pages.

## 🚀 Tecnologías Principales

- **Frontend**: React 19 (Hooks, Functional Components)
- **Estilo**: Tailwind CSS 4 con `@tailwindcss/vite`
- **Iconos**: Lucide React
- **Animaciones**: Motion (framer-motion)
- **Herramienta de Construcción**: Vite 6

## 📋 Características

- ✅ **Diseño Ultra Minimalista**: Enfocado en la legibilidad y la profesionalidad.
- ✅ **Optimización para PDF**: Incluye estilos específicos para impresión (A4/Carta).
- ✅ **Interactividad**: Efectos visuales suaves al interactuar.
- ✅ **Diseño Responsivo**: Se adapta perfectamente a dispositivos móviles y escritorio.
- ✅ **Datos Dinámicos**: Toda la información se gestiona desde un archivo de datos centralizado (`src/data.ts`).

## 🛠️ Instalación y Desarrollo Local

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/nombre-del-repo.git
    cd nombre-del-repo
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

4.  **Construir para producción:**
    ```bash
    npm run build
    ```

## 🌐 Despliegue en Cloudflare Pages

Para desplegar este proyecto en Cloudflare Pages:

1.  Conecta tu cuenta de **GitHub** a Cloudflare.
2.  Selecciona este repositorio.
3.  Configura los ajustes de construcción:
    - **Build Command**: `npm run build`
    - **Output Directory**: `dist`
4.  ¡Listo! Cloudflare se encargará del despliegue automático en cada cambio.
